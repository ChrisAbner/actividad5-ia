Para correrlo:

* Crear ambiente virtual: 

```bash 
$ python3 -m virtualenv venv
```

* Activar ambiente virtual: 

```bash 
$ source venv/bin/activate 
```
* Instalar dependencias: 

```bash 
$ pip install -r requirements.txt
```

* Correr script main:

```bash 
$ python3 main.py
```
